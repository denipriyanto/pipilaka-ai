# Pipilaka Project AI

Applikasi voice interaktif dengan karakter Pipilaka.

## Run Locally

Clone the project

```bash
  git clone https://github.com/denipriyanto/pipilaka-ai.git
```

Go to the project directory

```bash
  cd pipilaka-ai
```

Install dependencies

```bash
  pip install > requirements.txt
```

Start the server

```bash
  python3 app.py
```
